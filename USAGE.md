# DFN 使用指南

## 快速开始

### 1. 基本用法

```bash
# 为文档添加公式编号
python dfn.py example.docx output.docx
```

### 2. 使用章节号格式

```bash
# 默认模式：(1-1), (1-2), (2-1)...
python dfn.py thesis.docx output.docx
```

### 3. 使用连续编号

```bash
# Simple 模式：(1), (2), (3)...
python dfn.py thesis.docx output.docx --simple
```

### 4. 使用美观格式

```bash
# eqArr 格式（需要 t1.docx 模板）
python dfn.py thesis.docx output.docx --t1
```

## 前提条件检查

### 检查多级列表配置

1. 打开 Word 文档
2. 查看章节标题是否应用了"标题 1"样式
3. 查看是否有章节编号（如"第 1 章"、"第 2 章"）

如果没有，请：
1. 选中章节标题
2. 点击"开始"→"样式"→"标题 1"
3. 点击"开始"→"段落"→"多级列表"
4. 选择带章节号的格式

### 检查 t1.docx 模板

使用 `--t1` 参数前，确保 `t1.docx` 文件存在于：
- 脚本同一目录，或
- 输入文档同一目录

## 常见问题

### Q: 提示"文档未配置多级列表"
A: 文档的章节标题没有应用多级列表样式。请使用 `--simple` 参数跳过章节检测。

### Q: 公式编号后显示为乱码
A: 按 `F9` 更新所有域，或右键点击编号选择"更新域"。

### Q: 如何引用公式编号？
A: 在 Word 中使用"插入"→"交叉引用"→"书签"，选择对应的 `eq*` 书签。

## 输出示例

### 输入文档
```
第 1 章 引言

E = mc²

F = ma

第 2 章 相对论

p = mv
```

### 输出（默认模式）
```
第 1 章 引言

E = mc²#(1-1)  [书签：eq1_1]

F = ma#(1-2)   [书签：eq1_2]

第 2 章 相对论

p = mv#(2-1)   [书签：eq2_1]
```

### 输出（eqArr 格式）
公式编号显示为美观的数学格式，与公式融为一体。
